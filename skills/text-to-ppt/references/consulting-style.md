# Consulting Style

Use these rules for business, strategy, proposal, report, investor, or executive decks.

## Writing

- Use action titles that state the takeaway.
- Lead with the answer, then support it with bullets.
- Quantify claims where the source text provides numbers.
- Avoid generic labels like `Overview`, `Background`, and `Summary` unless they are section dividers.
- Keep one main message per slide.
- Use speaker notes for nuance, caveats, and narration.

## Visual Style

- Default background is white or near-white.
- Use black/dark gray text, one red accent, and light dividers.
- Avoid emoji, decorative icons, gradients, shadows, and unnecessary rounded cards.
- Use tables or simple comparison layouts when the source text has structured contrasts.
- Include a small source/footer when the input provides source context.

## Density

- Title: 28-34 pt.
- Body: 16-22 pt.
- Footer/source: 8-10 pt.
- Use no more than 5 bullets per slide.
- If a slide feels dense, split it instead of shrinking text aggressively.

## Validation Loop

After generating the deck:

1. Confirm slide count and order.
2. Confirm titles are insight-driven.
3. Confirm all text is readable.
4. Confirm the lower third is not awkwardly empty.
5. Confirm source/footer appears when required.
6. Regenerate after fixes.
